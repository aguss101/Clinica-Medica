Importante saber que este script aun tira algunos errores, es decir que hay varias
tablas que aun no llena con datos por completo.

El scriptV3 es la ultima version desarrollada para la insercion de datos en la 
base de datos ClinicaBD. Este es el archivo que se deben corregir sus errores.

El script que se encuentra dentro de la carpeta Borrado, es especificamente para que
en la prueba de corregir errores podamos hacer lo siguiente:

Probamos la ejecucion del scriptV3, cuando nos lanza error podemos ejecutar el 
script de borrado parae eliimnar todos los recientes insert con errores y volver
a intentar la ejecucion del script V3 con mejoras a los errores.

Corregir:

-Que se distinga el gentilicio segun si la persona a ingresar es hombre/mujer.

-Arreglar todos los mensajes de error que lanza la consola al ejecutar el script.
